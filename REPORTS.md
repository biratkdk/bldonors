# Reports Module

This module adds a basic reporting dashboard for the project.

## Features Added

- total donor count
- total blood request count
- total stock entry count
- total available blood units
- donor summary grouped by blood group
- blood request summary grouped by blood group
- stock summary grouped by blood group
- low stock list
- CSV export of the report

## URLs

- `/reports`
- `/reports/export/csv`

## Access

- login required

## Future Improvements

- monthly trend report
- graphical charts
- fulfilled vs pending requests
- printable report page
